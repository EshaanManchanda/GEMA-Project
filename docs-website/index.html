Gema Event Management Platform

A comprehensive, scalable event management platform designed to connect families with exciting kids' activities. Built with the MERN stack and featuring advanced role-based access control, payment processing, and multi-language support.

Quick Start Guide

Role-Based Access Control

Comprehensive user management with granular permissions for Admin, Vendor, Employee, and Customer roles

Event Management System

End-to-end event lifecycle management including creation, approval workflows, and booking systems

Payment Management

Multi-gateway payment processing with Stripe and PayPal, commission tracking, and automated payouts

Authentication System

Dual JWT and Firebase authentication with 4-digit OTP email verification for secure access
