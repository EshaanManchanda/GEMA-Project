 Enterprise Documentation Generation Plan

 Context

 Generate complete enterprise-level documentation for the Gema Event Management Platform directly from the codebase using available MCP tools. The project    
 has 49 models, 66 routes, 62 controllers, 43 services — too large for manual docs. MCP tools can scan AST and Mongoose schemas to produce accurate,
 code-grounded output.

 Existing docs: backend/docs/ has scattered markdown + Postman collection, but no structured ERD, SRS, security audit, flow diagrams, or test specs.

 Output target: backend/docs/generated/ (all MCP tools write there by default).

 ---
 Execution Plan
